# 📉 Customer Churn Prediction App

## 🚀 Overview
This is a Streamlit web application that predicts customer churn using a trained Logistic Regression model.

The app allows users to input customer details such as:
Demographics
Account information
Internet services
Billing details

It then calculates the probability of churn and classifies the customer as:

🚨 High Risk of Churning
✅ Low Risk of Churning

[Customer Churn Predictor Live App](https://custoomer-churn-logistic.streamlit.app/)

## 🧠 Model Details
Algorithm: Logistic Regression
Library: scikit-learn
Version: 1.6.1
Saved using: joblib

## 📊 Features
Dynamic form with dependency logic
Probability-based prediction
Custom churn threshold (0.35)
Clean UI with status indicators
Model version pinned for deployment stability
