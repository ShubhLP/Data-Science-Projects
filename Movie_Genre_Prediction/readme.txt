# Project Overview
Predict movie genres based on synopses using a fine-tuned BERT model! This app leverages natural language processing to classify movie synopses into multiple genres.

# Features
- Predict multiple genres for a single movie synopsis.
- User-friendly UI built with Streamlit.
- Real-time predictions with genre probabilities.
- Interactive and accessible from the web.

# Technologies
- Python: Core programming language.
- Streamlit: Web framework for interactive UI.
- Transformers (Hugging Face): Pre-trained BERT for text classification.
- PyTorch: Deep learning framework for model training.
- scikit-learn: MultiLabelBinarizer for multi-label genre transformation.
